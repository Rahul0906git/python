# Python Basics


**Complex Number**
```python
    z = 3 + 4j
    print(z)             # Output: (3+4j)
    print(z.real)        # Output: 3.0 (real part)
    print(z.imag)        # Output: 4.0 (imaginary part)
    print(abs(z))        # Output: 5.0 (magnitude = √(3² + 4²))
```
